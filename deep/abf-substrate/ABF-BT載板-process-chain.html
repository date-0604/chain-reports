1是什麼、漲價邏輯 高信心

「成本推動 + 缺貨拉動」雙因漲價循環

  • 需求拉動(規格升級):AI 晶片面積逼近 reticle、層數由 3+3 → 11+11 → 13+13,單顆 ABF 用量數倍升;且大尺寸/細線路良率低 → 更耗產能 → 放大缺口(高盛邏輯)。
  • 成本推動(上游漲價):味之素 ABF 膜(>95% 獨佔)在維權基金 Palliser 施壓下擬漲 ≥30%、MGC BT 樹脂與 CCL +30%(2026/4/1 生效)、T-glass 玻纖布交期拉長至 30 週+——載板廠取得對下游 ASP 漲價的正當性(2026 報價估 +15-20%、2Q26 約 +5%、味之素膜漲若實現可再傳導 3-6%)。
  • 財報已驗證(非純題材):欣興 1Q26 EPS 3.28 元創 13 季新高(稅後純益年增約 4.5 倍)、ABF 營收 QoQ +14%、Blackwell ABF 市佔逾 30%、美系 AI ASIC 載板市佔逾 70%。
  • 台灣的位置:IC 載板整體市佔台廠合計近半(欣興/南電/景碩),但最高階 AI 伺服器 FC-BGA(NVIDIA GPU 主載板)由日 Ibiden/Shinko 掌 70-80%;台廠在雲端 ASIC(Google/Meta/AWS)、網通、伺服器 CPU 載板地位較強。
切分(與已研究的鏈):本題聚焦「載板廠環節」(製程/產能/ASP/認證/獲利彈性);上游 T-Glass 玻纖布(材料端)與下一代 玻璃載板 GCS(製程端、2028-2030 接替)已各有專篇,本篇只交叉引用其向下傳導與接替時程,不重做。

2完整製程鏈拆解(材料 → FC-BGA)高信心

八環節:① 上游材料投入 → ② core/coreless → ③ 增層壓膜(ABF) → ④ 雷射鑽孔 → ⑤ mSAP 細線路 → ⑥ 表面處理 → ⑦ AOI/電測/翹曲管控 → ⑧ FC-BGA 封裝交界。台廠 alpha 在「載板製造本身的良率/產能/順價」,幾乎全數缺席於造設備、造膜。

1

上游材料投入(ABF 膜 / BT 樹脂 / 玻纖布 / 銅箔)⭐ 材料即瓶頸

備齊日系獨佔的關鍵材料:味之素 ABF 增層膜、MGC BT 樹脂、低 CTE/低介損 T-glass 玻纖布、極薄低稜銅箔——決定整片載板的電性、翹曲與良率天花板。

難點:材料配方(樹脂/填料/CTE/Df/Dk)直接決定後段所有良率與翹曲;ABF 膜+T-glass 均日系獨佔、擴產 2-3 年,載板廠無法垂直整合,只能被動接受漲價與配額。味之素 ABF 膜 >95% 獨佔、Q3-2026 擬漲 30%(維權基金施壓、缺第三方 benchmark)。
對應:味之素 ABF 膜(>95%)/ MGC BT 樹脂 / 日東紡 T-glass / 三井金屬 銅箔 台玻(T-glass,交叉引)/ 長春(銅箔) 高(獨佔結構)
2

核心層 core / coreless 設計成形

做出載板機械骨幹:有 core(玻纖補強硬板,翹曲易控、量產成熟)或 coreless(無核、薄、低電感、高佈線密度,AI 大尺寸傾向),並完成 core 層線路+鍍通孔(PTH)。

難點:coreless 缺玻纖剛性,峰值回焊溫度下翹曲變異比薄 core 高 25-50%(學術佐證);大尺寸(>100mm body、逼近 reticle)+多層下翹曲失控,carrier 貼合/剝離良率與對位是卡點。設計即鎖定後段翹曲上限。
對應:欣興 3037 / 南電 8046 / 景碩 3189 Ibiden(coreless 領先)/ Shinko / SEMCO / AT&S 高
3

增層 build-up(ABF 膜真空壓膜 lamination)

在 core 兩側交替疊上 ABF 膜+銅,用真空壓膜把絕緣層一層層長出來,堆出 11+11 甚至 13+13 高層數結構。

難點:ABF 膜本身味之素獨佔;壓膜需真空無氣泡、厚度均勻、與底層密合;層數越高累積應力與翹曲越大,任一層壓膜空洞(void)即報廢整片大面積板——void/缺陷是 AI 高階載板良率主因(學術)。
對應:欣興 / 南電 / 景碩 / 臻鼎(切入中) Ibiden / Shinko / SEMCO / AT&S;設備:名機/Bürkle 高
4

雷射鑽孔 / 微孔 via(UV / CO2)

用 CO2/UV 雷射在 ABF 膜上打出數萬個 <50μm 微盲孔(BMV),作為層間垂直互連通道,孔內 desmear 去膠渣後待化學銅/電鍍填孔。

難點:孔徑逼近 <50μm、孔密度達 2000 孔/inch²,要求孔形一致、無殘膠、孔底乾淨;單片大面積板數十萬孔,任一批孔缺陷即拉低整片良率。設備為瓶頸:14 層 ABF 交期 28 週、雷射鑽孔機 backlog 18 個月。
對應:欣興 / 南電 / 景碩(買機台做) 三菱電機 / Via Mechanics / LPKF(LPKF+三菱約 70%);Atotech(desmear) 高
5

細線路成形(mSAP / SAP 半加成 + 電鍍)⭐ 良率最敏感

用半加成法(mSAP L/S 達 ~25μm、SAP ~36μm pitch)在 ABF 上長出極細線路,搭配化學銅種子層、DI 曝光、電鍍銅填孔(void-free)、快速蝕刻,承載 AI 晶片暴增的 I/O 密度。

難點:L/S 微縮到 20/20μm 以下時,種子層快蝕過蝕(斷路)/欠蝕(短路)窗口極窄;大面積+多層使單層良率連乘後總良率快速下滑。能穩定做 20/20μm 以下者有限,構成台日韓技術分水嶺(味之素膜為共同料,差異化在 mSAP 製程能力)。
對應:欣興(高階 mSAP)/ 南電 / 景碩 Ibiden / Shinko / SEMCO;Atotech(藥水)/ Orbotech(DI)/ 旭化成(乾膜) 高
6

表面處理 / SMT pad / 平坦化

做最外層焊墊表面處理(ENEPIG/OSP)與防焊、平坦化,讓晶片端與主板端都能可靠焊接。

難點(相對成熟):高階 AI 載板 pad 間距縮小、共面度(coplanarity)要求高;表面處理層厚均勻性影響 FC bonding 良率;平坦化不足放大翹曲與焊接缺陷。技術門檻低於 mSAP/壓膜。
對應:欣興 / 南電 / 景碩(PCB 廠普遍能力) Atotech / 上村(表面處理藥水) 高
7

AOI 檢測 / 電測 / 翹曲 warpage 管控 ⭐ 頭號良率殺手

用 AOI/X-ray/電測逐片檢出開短路與孔缺陷,並全程量測與矯正大面積/高層數板的翹曲變形。

難點:大面積+高層數+coreless 使翹曲成為最難管控的良率瓶頸——不同 CTE 材料在回焊高溫翹曲導致橋接/枕頭效應(head-in-pillow);有機載板翹曲 >50μm@100mm(vs 玻璃 <20μm);檢測要在 5μm 線寬下抓微缺陷且不誤殺。CTE 匹配+退火矯正是勝負手。
對應:欣興 / 南電 / 景碩(內製 know-how);德律 TRI(電測,外銷) KLA/Orbotech・Camtek(AOI)/ Akrometrix(翹曲量測) 高
8

FC-BGA 封裝交界(OSAT / CoWoS)

把成品載板交給封測/晶圓代工端,作為 CoWoS-L/2.5D 先進封裝中承載中介層+HBM+GPU 的 FC-BGA 基座。

難點:大尺寸 AI 封裝(逼近 reticle、HBM 顆數增加)使載板尺寸/層數/翹曲與封裝端強耦合(co-design);載板翹曲若超規,在 FC bonding/回焊端直接造成封裝良率損失,是載板缺口傳導到 AI 算力出貨的最後關卡。「鎖產能不鎖價」長約鎖定載板廠 2027-28 獲利。
對應:台積電(CoWoS)/ 日月光(OSAT);欣興/南電/景碩(載板供應方) Ibiden/Shinko(供 Intel/AMD)/ Amkor 高

被對抗式查證否決/修正(請勿當既成事實):
  • 「味之素確定 Q3 漲 30%、高階 50%、台廠已確認」:此為媒體「市場傳出」口徑,源頭是英國維權基金 Palliser Capital 4 月入股後施壓味之素加價 30% 的訴求,並非味之素官方公告;原文用詞為「正考慮調漲至少 30%、最終價格仍在協商中」。無第三方獨立成交 benchmark;50% 一說僅見單一中國自媒體。判定待查,不採信為既成事實、亦不否定漲價趨勢
  • 「缺口就是 8/27/35%(或 10/21/42%、22%)可作統一既定數字」:三套口徑分屬不同機構且互不一致(高盛 2H26 10%/27 21%/28 42%;美銀匯豐 8/27/35;大摩 2030 22%;TPCA 2027 21-27%),屬外資/協會供需模型推估、未獨立驗證,不可混用為單一定論——應採區間並列
  • 「健鼎/華通/定穎是 ABF/BT 載板直接受惠載板股」:三者皆 PCB/HDI 廠,不生產 ABF/BT IC 載板(健鼎做伺服器/記憶體模組/車用 PCB、華通做 HDI/低軌衛星、定穎做車用/伺服器 PCB),屬環節分類錯置;其 AI/伺服器本業另有受惠,故不錯殺、維持觀察,但本題不應採信為直接受惠
  • 「欣興玻璃載板 GCS 將提早量產貢獻 2026」:欣興 GCS 為真實卡位但時程明確遞延——產線 2026 底裝機、2027-2028 量產,董事長曾子章自述材料未定案、技術未到位、多數客戶量產遞延至 2028 後。屬未來選擇權,2026 真受惠來自 ABF,不應把 GCS 計入當期

結構性風險與開放問題

  • 「吃漲價、也被上游漲價」:味之素 ABF 膜(>95%)+ MGC BT + T-glass 全在日本手上,載板廠無法垂直整合;轉嫁能力(能否全額順價)是毛利關鍵變數,缺第三方 benchmark
  • 高 beta 波動:南電 ABF 純度最高、漲價彈性最大,但反向波動也最劇烈(2024-25 曾因庫存調整獲利大幅下滑)
  • 擴產到位與 double-booking 風險:Ibiden(5,000 億)、SEMCO、深南新產能 2027 爬坡;若擴產超預期或缺口含 double-booking 雜訊,2028 峰值後恐回吐
  • 玻璃基板遠端替代:GCS 2028-2030 先侵蝕頂規,2028 前量級仍以 ABF 為主;須在 2027-2028 追蹤 GCS 良率/成本拐點是否提前,避免高基期追高被頂規流失反噬
  • 市場規模口徑陷阱:整體 ABF 載板 vs FC-BGA vs AI 伺服器子市場 vs ABF 膜材料,量級不可直接比;供需缺口三套口徑勿混報
  • 開放問題:味之素實際成交漲幅、台廠順價比例、Tesla ABF 訂單花落誰家(SEMCO vs LG Innotek)、玻璃接替拐點——均待後續追蹤
