1是什麼、雙循環邏輯 ＋ 兩大技術黑盒 高信心

漲價 + 規格升級雙循環

  • 規格升級:交換器/加速卡訊號速率 800G→1.6T,CCL 從 M7(2025 主流)→M8(2026 主流,800G/GB300)→M9(2026H2 小量、2027 放量,1.6T/CPO/Rubin mid-plane);每升一級 Df 遞降、ASP 跳升(M7≈$100-110 → M8≈$160-180 → M9 Q-glass 版 >$400,約 M7 的 4 倍)。
  • 漲價:2026Q2 CCL 漲價已落地(台光電 Q2 至少 +10%、高階供不應求);三井 MicroThin 銅箔 2026/4 漲 12%;HVLP3+ 加工費為一般銅箔約 10 倍、毛利 40-60%(Goldman)。
  • 台灣定價權最強:CCL 三雄掌全球高速 CCL 約 37.4%(台光電 18.9% 居冠);HVLP 銅箔金居切入二供(全球 HVLP4 已認證者僅三井/古河+金居等少數)。
🔒 黑盒 1:HVLP 銅箔「低粗糙度 vs 附著力」兩難 + 三井近壟斷
高頻下訊號集中於導體表面(1GHz 趨膚深度僅約 2.1μm),表面稜線(Rz)大於趨膚深度就增損耗,故須降粗糙度(STD 7.5μm→RTF 2.5μm→HVLP 1.6μm→HVLP4 ~0.5μm)。但銅箔靠「銅瘤+矽烷偶聯」與樹脂咬合產生剝離強度,降粗糙度=瘤變小=附著力下降。解法(各家最高機密):種比趨膚深度還小的微銅瘤+特殊矽烷鈍化(傳統用鎳會升損耗),窗口極窄。三井壟斷因=奈米銅晶粒控制+處理層配方多年 know-how+客戶認證綁定(良率僅 70-80%);金居憑 RTF/ARTF(進階反轉銅箔)know-how 切入二供。
🔒 黑盒 2:M8→M9 樹脂/填料配方 + 認證壁壘
傳統環氧達不到高頻低損,M8 大量導入 PPO/PPE+BMI,M9 反轉為碳氫樹脂為主(碳氫:PPO≈2:1)+加 BCB、最頂摻 PTFE;填料端高性能球形二氧化矽占比拉到 >40%(控 CTE/Df);補強材 M9 由 Low-Dk 玻纖布升級石英布 Q-glass(Dk≈2.3/Df≈0.0009)。三難同解:越低 Df 樹脂表面張力越低、越難黏住超平滑 HVLP4、還要高層數不翹。認證週期長(配方匹配→驗證→量產 >9 個月)、切換成本高→台光電靠首家量產認證取得領先定價權。
切分(與已研究的鏈):玻纖布/石英布「織造本身」見 T-Glass 玻纖布 專篇(本篇只引投料)、下游高層數 PCB/載板見 ABF/BT 載板 專篇(本篇止於 CCL 交貨點)。台燿/南亞跨 T-Glass 題,評受惠勿重複計。

2完整製程鏈拆解(布之後 → CCL 交貨)高信心

七環節:① 樹脂系統配方 → ② 玻纖布/石英布投料(交叉引 T-Glass)→ ③ HVLP 銅箔處理層(核心、黑盒1)→ ④ 含浸 prepreg → ⑤ 壓合 CCL → ⑥ 客戶認證 → ⑦ 下游高層數 PCB/載板交界。台廠 alpha 在 CCL 配方壓合與 HVLP 二供;設備端(含浸機/壓合機)台廠亦有 alpha。

1

樹脂系統配方(環氧 → PPO/PPE → 碳氫 → BMI)⭐ 黑盒2

把 PPO/PPE、碳氫樹脂、BMI、(頂階)PTFE 與液相法 SiO2 功能填料調成可含浸的樹脂膠水(varnish);M7→M9 隨 Df 遞降,碳氫樹脂用量大增(M9 碳氫:PPO≈2:1)、功能填料 >40%。

難點:超低損耗樹脂(PPE/PTFE)表面張力極低、黏合性極差,要同時做到低 Df 又黏得住銅、高溫壓合不爆板——配方(碳氫:PPO 比、填料種類/量)是各家最高機密,缺第三方 benchmark。
對應:台光電 2383 / 台燿 6274(配方自調) SABIC / 旭化成 / MGC(PPO);長春(台·未上市,樹脂) 高(機制)/ 中(填料逐項)
2

玻纖布 / 石英布投料(交叉引 T-Glass,不重做織造)

投入 Low-Dk 玻纖布(M7/M8)或石英布 Q-glass(M9)作補強材;Q-glass 近純 SiO2、Dk≈2.3/Df≈0.0009/CTE≤3,是 M9 mid-plane(傳達 104 層)翹曲控制關鍵。

難點:石英布開纖均勻度與含膠浸潤難;上游高階 Low-Dk 布日東紡 Nittobo 近 90% 壟斷+缺料(織造製程見 T-Glass 專篇)。
對應:台玻 1802 / 富喬 1815 / 建榮 5340 / 德宏 5475(石英) 日東紡 Nittobo / 旭化成・信越(石英) 高(見 T-Glass 篇)
3

HVLP 銅箔製造:電解生箔 + 粗化/處理層 ⭐ 整鏈最卡(黑盒1)

先電解生箔(銅離子電鍍到旋轉陰極輪),再做多道表面處理(結瘤粗化、固化、鈍化、矽烷偶聯);把霧面 Rz 從 RTF ~2.5μm 壓到 HVLP ~1.6μm、HVLP4 ~0.5μm。HVLP3=PCIe6/800G、HVLP4=PCIe7/NVLink5。

難點:低 Rz 必然犧牲剝離強度(低粗糙度 vs 附著力物理矛盾);解法是種比趨膚深度還小的微銅瘤+特殊矽烷鈍化,窗口極窄、為各家最高機密。三井最高階 HVLP4 近壟斷,整鏈最卡關環節(良率僅 70-80%)。
對應:金居 8358(HVLP4 二供,ARTF)/ 南亞銅箔 1303(HVLP3 量產) 三井金屬 / 古河 / 福田 / 日本電解 / Solus(韓) 高
4

含浸 / 塗膠(樹脂含浸玻纖布 → prepreg 黏合片)

玻纖布/石英布浸入樹脂槽含浸,經烘箱去溶劑、樹脂半固化(B-stage),收捲成 prepreg(PP)。

難點:B-stage 硬化程度、含膠量均勻、溶劑殘留直接決定後段壓合良率與電性;低 Df 樹脂(碳氫/PPO)流動性差更難浸潤、含浸窗口窄。含浸機交期已拉長至 1-2 年(設備瓶頸,台廠亞泰金屬 AMI 受惠)。
對應:台光電 / 台燿 / 聯茂;亞泰金屬 AMI(含浸機 >70%) Panasonic / Doosan(韓) 高
5

壓合 Lamination(prepreg + HVLP 銅箔 → CCL)

prepreg 疊層+HVLP 銅箔,經真空高溫高壓壓合,樹脂完全固化(C-stage)、與銅箔黏合成銅箔基板(CCL)。

難點:高溫壓合時低損耗樹脂與超平滑銅箔最易分層(delamination)/爆板;Q-glass/Low-CTE 與銅 CTE 不匹配造成翹曲(IPC 要求 ≤0.75%);熱壓窗口窄(良率挑戰最高)。設備台廠志聖/群翊受惠。
對應:台光電 / 台燿 / 聯茂;志聖 3567・群翊 6664(壓合設備) Bürkle(德)/ 北川精機(日) 高
6

客戶認證(M8/M9 認證壁壘、週期)⭐ 龍頭護城河

量產級 CCL 送 PCB 板廠+終端(輝達/CSP)做電性、信賴性、製程相容性驗證,通過才進 AVL 合格供應商名單、取得 design-win。

難點:認證需多季～逾一年(聯茂 2025H2 過認證、2026 才逐步貢獻營收);通過後切換成本極高、GPU 大廠不輕易換料,形成龍頭護城河。是台光電 M9「先卡先贏」根本(但聯茂 M9 亦已過認證,非唯一持證者)。
對應:台光電(首家量產 M9 供輝達)/ 聯茂(M9 過認證待量產)/ 台燿(M8) NVIDIA / AWS / Google / Meta(認證方) 高
7

下游高層數 PCB / 載板交界(只做對照)

CCL 出貨給 PCB 板廠做高層數主板(M9 mid-plane 傳達 104 層),或載板廠做 ABF/BT;本報告止於 CCL 交貨點。

難點:層數越高,壓合與鑽孔對位累積誤差越大;CCL 板材一致性直接影響 PCB 良率。下游製程見 ABF/BT 載板專篇。
對應:臻鼎 4958 / 金像電 2368 / 華通 2313 / 欣興 3037 / 南電 8046 Ibiden / Shinko(載板) 中(交界對照)
