1是什麼、為何卡住台灣 高信心

CPO 的本質:把「光」搬進晶片封裝

傳統交換器把 SerDes 訊號拉到前面板插上可插拔光模組(pluggable);CPO(Co-Packaged Optics 共封裝光學)把光引擎(完成電→光轉換的矽光子晶片)搬到交換 ASIC 旁、同一封裝內,縮短電氣路徑、大降功耗。NVIDIA 自述較可插拔省電 3.5x、Broadcom 自述光互連省電 70%(皆公司自述、缺第三方 benchmark)。

  • 為何台灣卡得住:CPO 光引擎=PIC(矽光子晶片,含 200G 微環調變器 MRM/波導)+ EIC(電子驅動晶片),用台積電 COUPE 平台以 SoIC-X 垂直堆疊、再進 CoWoS 與 ASIC 共封。NVIDIA 與 Broadcom 的 CPO 都建在 COUPE 上——台灣在「晶圓級光電整合」這層近乎主導、無直接競爭者。
  • 但價值量最大的光引擎設計 IP(MRM/MZM 調變器、架構)仍握在 NVIDIA/Broadcom 手上,台積電吃晶圓代工與封裝整合;台股設計端著墨不到調變器 IP 本身。
  • 三大失效/瓶頸物理:(1) MRM 微環調變器對溫度極敏感(每 +1°C 共振波長漂移約 0.1nm),須加熱器閉迴路鎖波長、對抗多環熱串擾;(2) FAU 光纖陣列對準容差是 sub-micron,SemiAnalysis 點名為良率瓶頸;(3) 雷射是整機最高失效率元件,故業界一致把雷射「外置」做成可插拔 FRU(ELS)。
切分(與已研究的鏈):本題講「光的路徑」(雷射光源 → PIC/EIC 光引擎 → FAU 耦合 → 封測 → 系統);與「InP 光通訊上游」(基板/磊晶/雷射晶粒材料,上游)、「玻璃載板 GCS」(基板材料)相鄰但不同。聯亞/全新跨 InP 題、日月光跨 GCS 題,評估受惠勿重複計;「CPO 下一代基板可能採玻璃中介層」僅作介面交代,基板製程見 GCS 報告。

2完整製程鏈拆解(架構 → 系統)高信心

五環節:① 交換 ASIC＋CPO 架構定義 → ② COUPE 光引擎晶圓(PIC/EIC・SoIC-X,台灣最硬)→ ③ 雷射光源 ELS(InP,料源台廠卡位、整模組缺席)→ ④ FAU 光纖陣列＋晶圓級耦合(上詮卡位、機台缺席)→ ⑤ 光引擎組裝＋MRM 熱鎖＋封測＋系統(日月光/訊芯)。

1

交換 ASIC 與 CPO 系統架構定義

把交換晶片(NVIDIA Quantum-X/Spectrum-X、Broadcom TH6-Davisson 102.4T)的 SerDes 從「拉到前面板可插拔」改成「光引擎搬到 ASIC 旁同封裝」;定義幾顆光引擎、每顆速率、走 MRM/MZM 哪種調變、雷射是否外置可拆。

難點:系統級取捨——電氣/熱/可維修性與光學良率互相拉扯。CPO 把「一個光埠壞了不能像插拔模組那樣換」變成系統風險,架構期就須決定雷射外置與光引擎做成可拆 sub-assembly。TH6-Davisson=16 顆 6.4T DR 光引擎拼成 102.4T(官方、已出貨),概念已成熟。
對應:NVIDIA / Broadcom / Marvell / Cisco 台積電 2330(ASIC＋CoWoS 代工) 高
2

COUPE 光引擎晶圓製造(PIC＋EIC・SoIC-X 堆疊・200G MRM)⭐ 台灣最不可替代

晶圓廠把 65nm SOI 矽光子晶片 PIC(含 200G 微環調變器 MRM、波導、光柵)與 6nm 電子驅動晶片 EIC,以 SoIC-X 混合鍵合垂直堆疊成單一光引擎,再進 CoWoS 與 ASIC 共封。COUPE=SoIC-X(PIC/EIC)+ CoWoS。

難點:把 6nm CMOS 與 65nm 矽光子兩個製程世代用 SoIC-X 晶圓級混合鍵合,需 sub-micron overlay 對位＋極低介面阻抗;PIC 趨向超薄大面積,鍵合前晶圓翹曲/平整度直接吃掉 overlay 良率。MRM 製程變異使共振波長偏移,KGD 良率端要靠後段熱調校救。2026 首度大規模商轉,存在 ramp 風險。
對應:台積電 2330(COUPE,獨家) GlobalFoundries Fotonix / Intel / Tower / Ayar Labs+Alchip 高(2026 量產屬首度商轉、ramp 中信心)
3

雷射光源 ELS 製造(InP 磊晶 → CW DFB / EML → 外置光源模組)

在 InP 基板上長磊晶、做成連續波 CW DFB 雷射(給 MRM 矽光子的外置光源)或直調 EML,封成外置雷射模組 ELS/ELSFP,把光「餵」進光引擎。雷射晶片(發光體)只能做在 InP 上、無矽替代。

難點:reliability 與高功率壽命——高輸出雷射發熱大、熱應力下衰退快,雷射＋TEC 約占 ELS 70% 功耗;CW 需窄線寬、低 RIN、波長穩定才餵得動溫度敏感的 MRM。InP 磊晶門檻高、6 吋基板供應受限。整模組由 Lumentum/Coherent 寡占(NVIDIA 2026/03 各注資約 $20 億鎖產能);台廠在晶粒/磊晶料源卡位、整模組本體缺席。
對應:Lumentum(唯一量產 200G/lane EML)/ Coherent / 古河 / 三菱 / 住友 聯亞 3081(CW/EML 磊晶晶粒) 高(整模組寡占);台廠料源中信心
4

FAU 光纖陣列製作＋晶圓級耦合對準(sub-micron・耐迴流焊)⭐ 整鏈最卡良率

把多根光纖排成精密陣列 FAU(2D/1D、玻璃 V-groove),與 PIC 波導/光柵做 sub-micron 主動/被動對準,UV 膠固著,確保低插損且能撐過後段 260°C 迴流焊。

難點:對準容差 sub-micron,一偏移插損暴增;HVM 要兼顧精度＋吞吐＋(可拆卸式)多次插拔重複性;固著後「耐迴流焊」——封裝高溫讓 UV 膠與 PIC 翹曲改變對位、熱循環下漂移。上詮以 ReLFACon 專利平台降低高溫封裝耦合損耗,是其護城河。注意:晶圓級耦合「機台」本身由德商 ficonTEC/PI 主導,台廠是用機台做 FAU、非造機。
對應:上詮 3363(FAU＋與台積共研對準,核心) 波若威 3163 SENKO / 住友 / Corning / 天孚 TFC(陸)/ ficonTEC・PI(機台) 高
5

光引擎組裝＋MRM 熱鎖＋封測＋系統整合

把 COUPE 光引擎、FAU、外置雷射 ELS、ASIC 與散熱整合進封裝/系統,建立 MRM 閉迴路熱波長鎖定,做光電聯合測試與可靠度驗證後出貨 CPO 交換器(如 51.2T/102.4T)。

難點:MRM 對溫度極敏感(0.1nm/°C 漂移＋環間熱串擾),須整合加熱器＋光功率監測＋閉迴路控制器即時鎖波長,在高溫高密度環境最棘手;異質整合多元件、良率連乘(KGD↘);嵌入封裝的光埠失效難現場更換,須把光引擎/雷射做成可拆 FRU。邊緣耦合 vs 光柵耦合對準貼合,260°C 迴流焊。
對應:日月光投控 3711(VIPack 後段整合) 訊芯-KY 6451(edge-coupling CPO 封裝,Broadcom) 華星光 4979(模組端) ASE/SPIL / Fabrinet / 旭創・Eoptolink(陸) 高

被對抗式查證否決/修正(請勿當既成事實):
  • 上詮 3363「台積電 COUPE 前兩代獨家 FAU 供應商」:上詮法說只確認其為「關鍵 FAU 供應商」、客戶 82% 美國但未具名,從未自稱「獨家/前兩代獨家」——此措辭為媒體彙整/法人推估。FAU 角色真實(維持中-高),但「獨家」應標待查、不可寫死
  • 日月光 3711「CPO 已量產＋六座新廠」:日月光自述 CPO 仍早期開發、FOPLP 2027 才小量量產——「已量產」與進度不符。capex 70 億美元、先進封裝倍增、買群創南科五廠/聯合再生竹南廠為真;「六座新廠」缺單一官方清單、屬擴產敘事彙整。當期真受惠來自 CoWoS/SiP 大盤,非 CPO
  • 波若威 3163「已正式進入 NVIDIA/Google CPO 供應鏈(design-win)」:查證僅到「被視為潛在合作夥伴、CPO 產品 2025H2 完成驗證、2026 放量」與「CPO FAU 尚無統一標準」,未見確認 design-win 或客戶具名。應標「潛在/驗證中」,但不否定其 FAU/MPO 與 AI 光纖套件真實受惠
  • 全新 2455 與聯亞 3081 同為「CW laser / CPO 直接受惠」:兩者皆磊晶料源端而非 CPO 成品。聯亞較貼近 CW laser 晶粒+InP 磊晶;全新偏 800G PD 與 LD 發射端 GaAs 磊晶,與 CPO 所需 InP CW DFB 不同源。把兩者一律貼「CPO 直接受惠」是過度外推
  • 華星光 4979「2026 CPO 大爆發、CPO 王者」:華星光 CPO 切入點為 ELS 與 Fiber Attach,法人明確指 CPO 增量落 2027,2026 主受惠仍為 800G/1.6T 模組。「2026 大爆發」為題材前置
  • IET-KY 4971 為「CPO 核心受惠股」:MBE 磊晶稀缺性為真,但 CPO 主流光源多採 MOCVD-based InP/GaAs,MBE 對 CPO 直接料源路徑佐證較弱;與光聖換股「共同開發 CPO」屬早期、營收貢獻待查。維持觀察
  • 天孚通信 TFC「FAU 獨家供應」:來源為陸資投資部落格、缺第三方 benchmark;NVIDIA 官方生態系名單同列波若威/SENKO/Corning,顯示 FAU 非單一獨家
