1是什麼、製程鏈分層 ＋ 兩大判斷 高信心

製程鏈與本題範圍

  • 製程鏈:2.5D/3D 先進封裝逐站=① 矽中介層/RDL 重布線 → ② TSV 矽穿孔 → ③ 凸塊 bump/microbump → ④ CoW(Chip-on-Wafer)鍵合 → ⑤ molding 塑封 → ⑥ 壓力除泡/翹曲控制 → ⑦ 磨切 grinding/dicing → ⑧ 檢測/量測。CoWoS 分 -S(全矽中介層,密度最高、上限約 3.3X reticle)/-R(有機 RDL)/-L(局部矽橋 LSI+RDL,超大封裝、Rubin 達 4X reticle、掛 12-24 顆 HBM)。
  • 典範轉移(核心增量):晶圓級(CoWoS,12 吋圓)→ 面板級(CoPoS/FOPLP,矩形大面板,分階段 310×310→515×510→750×620mm)。面積利用率大增(515×510=12 吋的 4.5 倍、600×600=6 倍)、可做超大封裝,但整套曝光/塗佈/烘烤/檢測設備重新洗牌、翹曲控制變極難。
  • 台廠定位:台積電「主導」全球 2.5D/3D 高階產能、委外溢出由矽品(日月光投控)承接;設備端台廠卡濕製程(弘塑/辛耘)、壓力除泡烘烤(印能/志聖/群翊)、點膠貼合檢測(萬潤/均華/由田/牧德)、die bonder(均華),核心曝光/鍵合/磨切/塑封仍美日歐;FOPLP OSAT(力成/日月光/群創)台灣進度領先韓中。
  • 切分:材料端(玻璃基板 GCS/T-Glass/ABF/CCL/底膠塑封料)已各自獨立研究、本題只做一頁銜接、不重複計受惠主體(味之素/Resonac/JSR 主導底膠/塑封/介電,台廠材料端陪跑);本題從「設備 capex 鏈+OSAT+台積擴產時程」角度補強製程站別與訂單佐證。
🔍 判斷 1:現在式(CoWoS 擴產確定)vs 未來式(CoPoS 選擇權)——時序與確定性必須拆開
把「CoWoS 現在擴產」與「CoPoS 未來典範轉移」分層看,是本題對投資定位最重要的區分。短中期(2026-2027)=確定性題:台積 CoWoS 擴約 5 成(80K→120-130K)+委外 24-27 萬片是已下單的硬擴產、能見度到 2027,台廠設備股「當下的營收」幾乎全來自此(印能前 5 月 +83%、弘塑 2025 出機 +70-100% 是實質落地非概念)。中長期(2028-2029)=選擇權題:晶圓→面板級 CoPoS/FOPLP 確給台廠卡進更核心環節的機會(面板設備經驗+翹曲控制),但魏哲家 6/4「2-3 年沒捷徑」定調量產 2028 底-2029、良率未驗證、市場規模小且口徑分歧。鐵則:勿把 2028-29 前瞻當現況灌估值。
🔍 判斷 2:設備價值量天花板——核心進口(美日歐)vs 台廠周邊(1-2 成)
先進封裝設備資本支出兩極化。核心高門檻、價值量大、台廠吃不到:曝光(ASML/Nikon/Canon)、電鍍/CMP(Applied/Lam)、hybrid bonding/die bonder(BESI/ASMPT/EVG/SUSS/Shibaura/K&S)、磨切(DISCO 近獨占)、塑封(TOWA)、精密量測(KLA/Onto)。台廠卡周邊:濕製程(弘塑/辛耘)、壓力除泡烘烤(印能/志聖/群翊)、點膠貼合檢測(萬潤/均華/由田/牧德),媒體估占先進封裝設備 capex 約 1-2 成、無硬數字,贏在客製化/價格/在地服務而非技術主導。晶圓→面板化是台廠唯一的結構性升級窗口(面板/PCB 設備 know-how 可移植),但放量在 2027 後設備招標、量產 2028-29。

2製程鏈拆解(中介層 → 檢測,8 站)高信心

八站:① 矽中介層/RDL → ② TSV 矽穿孔 → ③ 凸塊 bump → ④ CoW 鍵合 → ⑤ molding 塑封 → ⑥ 壓力除泡/翹曲 → ⑦ 磨切 → ⑧ 檢測。台廠 alpha 在 ⑥(壓力除泡/翹曲,印能自述市佔約 80%)與 ⑧(檢測 AOI 供應鏈完整)、面板級的 ⑤(真空層壓);難度最高的 ① 中介層(曝光/電鍍/CMP 全進口、die shift/翹曲)與 ⑦ 磨切(DISCO 近獨占)台廠陪跑。

1

① 矽中介層 / RDL 重布線 — 全鏈技術密度最高、台廠最淺

先進封裝的「地基」:在 12 吋晶圓(CoWoS-S)或 RDL 載板(-R/-L)上建多層金屬重布線,把上方多顆 GPU/HBM die 的數萬條 I/O 重新繞線到下方基板。含光阻塗佈/曝光顯影/電鍍銅/CMP 多次堆疊。-S 全矽密度最高(上限約 3.3X reticle、~2700mm²)、-L 嵌局部矽橋突破 reticle 做超大封裝。

難點(全鏈最高=20/20):多層微影+電鍍+CMP 堆疊、最細線寬;設備自製度對台廠最低(曝光 ASML/Nikon、電鍍/CMP Applied/Lam/Ebara 全進口)。面板化雖開升級窗口,但 die shift/翹曲導致對位漂移是面板級最難啃的良率殺手(群創 RDL-first 2026 才拚認證)。台廠僅卡濕製程清洗周邊(弘塑/辛耘),核心微影/電鍍/CMP 缺席。
對應:弘塑 3131 / 辛耘 3583(濕製程清洗周邊);面板級 RDL:群創 3481 / 志聖 2467 / 群翊 6664 ASML / Nikon / Canon / Applied / Lam / TEL / SCREEN / Ebara 高
2

② TSV 矽穿孔(蝕刻+電鍍填孔)— 核心進口、台廠卡清洗周邊

在矽中介層/晶片上垂直鑽微米級深孔並填銅,做 3D 垂直互連的「電梯井」。流程:深矽蝕刻(Bosch DRIE)挖孔→介電/阻障沉積→電鍍銅填孔→CMP 磨平→背面薄化露出 TSV(reveal)。孔徑越小、深寬比越大,挖孔與無孔洞填銅越難。面板級對應走 TGV(玻璃穿孔、雷射鑽孔)=屬玻璃基板供應鏈(已另研究)。

難點(高=17/20):高深寬比挖孔+無孔洞填銅+背面 reveal 薄化,核心 DRIE/電鍍/CMP 全在 Lam/Applied/Ebara/DISCO;面板級走 TGV「換跑道」但屬玻璃基板供應鏈、本題不重複計。台廠僅卡濕製程清洗(弘塑)、再生晶圓與部分濕蝕刻(辛耘)、缺陷檢測周邊。
對應:弘塑 3131 / 辛耘 3583(濕製程/再生晶圓周邊) Lam / Applied / KLA-SPTS / TEL / Ebara / DISCO / ACCRETECH 中高
3

③ 凸塊 Bump / Microbump — 周邊吃得到、核心吃不到

在晶片/中介層接點長出焊接凸塊做 die 對 die、die 對中介層的連接點。流程:UBM 濺鍍→光阻定義→電鍍銅柱(Cu pillar)+焊料(SnAg)→去光阻→回焊成球。AI 大晶片 microbump 上看數十萬顆、間距(pitch)微縮到 40μm 以下,是邁向 hybrid bonding(無凸塊銅對銅直接鍵合)的過渡。凸塊均勻度與共面性直接決定下一站鍵合良率。

難點(中高=15/20):凸塊電鍍核心設備在 Lam/Applied/ASMPT(NEXX),但 microbump 微縮與面板級大面積均勻度是真難題。台廠卡濕製程清洗去膠(弘塑/辛耘)、凸塊 AOI 檢測(由田/牧德)、回焊除泡(印能);面板化放大檢測與除泡需求對台廠正向。
對應:弘塑 3131 / 辛耘 3583 / 印能 7734 / 由田 / 牧德 3563(周邊濕製程+檢測+除泡) Lam / Applied / ASMPT(NEXX) / TEL / SCREEN / BTU / KLA / Onto 中高
4

④ CoW Chip-on-Wafer 鍵合 / hybrid bonding — 對準精度最高、均華卡取放

把已測試良好的 GPU/HBM 晶片一顆顆精準放到中介層/晶圓上並鍵合(CoWoS 的「Chip-on-Wafer」)。傳統熱壓鍵合(TCB:凸塊+焊料+加壓加熱);次世代 hybrid bonding(銅對銅+介電對介電直接鍵合、無凸塊、間距<10μm、頻寬密度躍升,SoIC/3D 堆疊用)。對位精度次微米級(<1μm),是全鏈「取放+對準」精度最高的設備站、直接決定 AI 大晶片良率。含暫時鍵合/解鍵合(載板供薄化)。

難點(高=18/20):次微米對位+hybrid bonding 整機由 BESI(Applied 持股 9%)/EVG/SUSS/ASMPT/Shibaura/K&S 壟斷,台廠難自製整機。台廠最有機會=均華 6640(高精度 die bonder/取放,台積 chip sorter 自述市佔逾 95%、G2C+ 聯盟負責面板級 chip-on-panel 鍵合)+辛耘 3583(暫時鍵合載板/解鍵合服務);面板化是均華卡進更核心環節的跳板。
對應:均華 6640(die bonder/取放) / 辛耘 3583(暫時鍵合載板) BESI / EVG / SUSS / ASMPT / Shibaura / K&S 高
5

⑤ Molding 塑封 / 真空層壓 — 晶圓級進口、面板級台廠有實質卡位

用環氧塑封料(EMC)包覆鍵合好的晶片提供機械/散熱/絕緣保護,並決定封裝體翹曲行為。晶圓級用壓縮成型(compression molding);面板級 FOPLP 多用真空層壓把樹脂膜壓合到大面板。填充均勻度、無孔洞(void)、CTE 匹配是控制下游翹曲的源頭。塑封料(底膠/EMC)材料端由日商(味之素/Resonac)主導=屬另研究、本站不展開材料。

難點(中=13/20):晶圓級壓縮成型整機由 TOWA/ASMPT 主導,台廠較難切;但 面板級真空層壓是台廠強項——志聖 2467(G2C+ 負責大面積真空層壓+熱烘烤)、印能 7734(真空壓合除泡)、萬潤 6187(底膠點膠);這套 know-how 與顯示器/PCB 層壓相通,台廠有面板產業經驗優勢。
對應:志聖 2467 / 印能 7734 / 萬潤 6187 / 群翊 6664(面板級真空層壓/點膠) TOWA / Apic Yamada / ASMPT;材料(另研究)味之素 / Resonac 中高
6

⑥ 壓力除泡 / 翹曲控制 ⭐ 台廠卡位最深、定價權最強

封裝過程中底膠填充、塑封、回焊都會夾帶氣泡(void)並因 CTE 不匹配產生翹曲(warpage),用高溫高壓烤箱(壓力除泡爐)把氣泡壓潰排出、同步控制翹曲與助焊劑殘留。這是 AI 大晶片良率的隱形關卡——晶片越大、孔洞與翹曲越致命。印能科技 7734 自述全球先進封裝除泡設備市佔約 8 成,是台廠在先進封裝設備鏈中市佔最高、最具話語權的單一環節。

難點(對台廠最有利=12/20):面板級翹曲控制是典範轉移最難工程問題之一(大面板剛性低、CTE 失配下翹曲被面積放大、角落最嚴重),但 台廠自製度與卡位深度最高——印能除泡自述市佔約 80%(2026 前 5 月營收 +83%、訂單到 2026 底)、第四代 EvoRTS 獲 2025 R&D 100 Awards(亞洲唯一);志聖/群翊熱烘烤、均豪 5443 翹曲量測;G2C+ 聯盟整套衝著面板級翹曲控制而來=「難題即商機」。
對應:印能 7734(除泡,自述市佔約 80%) / 志聖 2467 / 群翊 6664 / 均豪 5443(翹曲量測) (此環節國際對手相對分散、台廠印能為自述市佔領導者) 中高
7

⑦ 磨切 Grinding / Dicing(薄化與切割)— DISCO 近獨占、台廠陪跑

封裝完成後把晶圓/面板背面研磨減薄(thinning,讓 TSV reveal、降厚度與翹曲),再切割(dicing)成單顆封裝體。AI 大晶片要求超薄化(<50μm)又不能破裂、切割道極窄又要零崩裂(chipping),是機械精度與良率硬仗。DISCO 在薄化與切割幾近獨占、ACCRETECH 居次,Yole 估薄化設備市場 2025 約 5.82 億美元→2030 約 8.45 億美元。

難點(高=18/20):超薄化+零崩裂切割機械門檻高、DISCO 近獨占使台廠自製度極低、卡位淺(僅量測/周邊);面板化雖開量測機會但核心整機仍進口。台廠卡面板級磨切量測(均豪 5443 G2C+ 分工)、雷射切割(鈦昇)、磨切前後清洗(弘塑/辛耘)。
對應:均豪 5443(磨切量測) / 弘塑 3131 / 辛耘 3583 / 鈦昇(雷射切割) DISCO(近獨占) / ACCRETECH 中高
8

⑧ 檢測 / 量測(AOI/X-ray/SAM)⭐ 台廠卡位第二深

貫穿全鏈的良率守門員:RDL 線寬/疊對量測、凸塊共面性 3D 量測、鍵合對位檢查、翹曲量測、塑封 void/裂紋檢測(AOI/X-ray/超音波 SAM)、最終電性測試。AI 大晶片單價極高、一顆報廢損失大,先進封裝把「早檢、勤檢、面板級量越多」推到極致。

難點(對台廠較友善=11/20):高階缺陷量測(KLA/Onto)門檻高仍進口,但 台廠在 AOI/光學檢測卡位深(由田/牧德/萬潤完整供應鏈);面板化是「量的爆發」——面板越大、單片待檢 die/凸塊/RDL 越多,AOI 設備吞吐量與台數需求倍增=最即時、最確定的台廠受惠;面板級檢測量測由均豪 5443(G2C+)承接。
對應:萬潤 6187 / 由田 / 牧德 3563 / 均華 6640 / 均豪 5443(AOI/3D/面板級量測) KLA / Onto / CyberOptics / Nordson / Sonoscan 中高

裁決(verdict):分層判斷——把「CoWoS 現在擴產」與「CoPoS 未來選擇權」拆開看。【短中期(2026-2027)=確定性題】台積 CoWoS 擴約 5 成+委外 24-27 萬片是已下單的硬擴產、能見度到 2027,受惠最實的是濕製程(弘塑/辛耘)、壓力除泡烘烤(印能/志聖)、點膠檢測(萬潤/均華)、委外 OSAT(矽品/日月光)、FOPLP chip-first(群創已量產),但此段共識已部分反映在估值、追高需看交期與毛利轉換。【中長期(2028-2029)=選擇權題】晶圓→面板級 CoPoS/FOPLP 確給台廠卡進更核心環節的機會(面板設備經驗+翹曲控制),但「2-3 年沒捷徑」定調量產 2028 底-2029、良率未驗證、市場規模小且口徑分歧=尚未兌現的前瞻價值。【鐵則不變】先進封裝設備的核心曝光/鍵合/磨切/塑封仍美日歐、台廠占設備支出約 1-2 成、贏在客製與在地;材料端(玻璃/T-Glass/ABF/CCL)已另研究不重複計。結論:CoWoS 是「現在式的確定擴產」、CoPoS/FOPLP 是「未來式的台廠升級選擇權」,兩者都真但時序與確定性不同,勿把 2028-29 前瞻當現況灌估值。
